## Formula Reference

```
extended_price       = quantity × unit_price
line_discount_amount = extended_price × line_discount_percent / 100    [rounded 2 dp]
line_net_price       = extended_price − line_discount_amount           [rounded 2 dp]

total_price          = Σ extended_price
total_discount       = Σ line_discount_amount
vat_amount           = (total_price − total_discount) × vat_rate       [rounded 2 dp]
amount_due           = (total_price − total_discount) + vat_amount
```

> **Common mistakes:**
> - Forgetting rounding → totals may be slightly wrong
> - Making discount fields editable → wrong design
> - Not sending `line_discount_percent` in the payload → value lost on save
> - Hardcoding vat_percent instead of reading from DB → not flexible
